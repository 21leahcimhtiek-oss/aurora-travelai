# WanderAI

> AI travel planner and itinerary builder

## Features
- Itinerary generation
- Budget tracker
- Hotel recommendations
- Local tips
- Packing lists

## Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- OpenAI GPT-4o-mini
- Stripe Subscriptions
- Vercel deployment

## Quick Start

```bash
npm install
cp .env.example .env.local
# Fill in API keys
npm run dev
```
